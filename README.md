node-sgf-parser
===============

A node.js sgf files parser.
Based on [wgo.js](https://github.com/waltheri/wgo.js).

Installation
============
```
npm install node-sgf-parser
```

Usage
=====
```
var sgf_parser= require('node-sgf-parser');
var sgf = sgf_parser.parseFromSgf(your_sgf_file_string); 
//do stuff
```
